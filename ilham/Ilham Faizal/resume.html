JAWABAN NOMER 1

Cara kerja website dapat dijelaskan secara sederhana sebagai berikut: Pembuatan website Website dibuat dengan menggunakan bahasa pemrograman web, seperti HTML, CSS, dan JavaScript. Bahasa pemrograman ini digunakan untuk menyusun halaman-halaman website, seperti halaman beranda, halaman produk, halaman artikel, dan sebagainya. Penyimpanan website Setelah website dibuat, data-data website tersebut perlu disimpan di suatu tempat yang disebut web hosting. Web hosting adalah layanan yang menyediakan ruang penyimpanan data website di server. Pemasangan domain Domain adalah alamat website yang digunakan untuk mengakses website tersebut. Domain biasanya terdiri dari dua bagian, yaitu nama domain dan ekstensi domain. Nama domain adalah bagian yang dapat diingat oleh pengguna, sedangkan ekstensi domain menunjukkan jenis domain tersebut, seperti .com, .co.id, .org, dan sebagainya. Pengaksesan website Pengguna dapat mengakses website dengan mengetikkan nama domain website tersebut di address bar web browser. Web browser akan mengirimkan permintaan ke web server tempat website tersebut disimpan. Web server kemudian akan mengirimkan data-data website tersebut ke web browser. Web browser akan menampilkan data-data tersebut kepada pengguna.
